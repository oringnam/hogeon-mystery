# 5단계 분기형 구조 설계

## 경로 1: 디지털 흔적 (SNS/온라인)

### 1단계: 조사 방향
- "SNS 계정을 파보자"

### 2단계: 플랫폼 선택 (3개)
- 페이스북 타임라인 분석
- 인스타그램/블로그 확인
- 온라인 커뮤니티 활동

### 3단계: 특정 게시물 발견 (3개)
- "인생 역전" 포스팅 (→ 로또 방향)
- 이세계 소설 리뷰 (→ 판타지 방향)
- 헬스장 인증샷 (→ 스파이 방향)

### 4단계: 심화 분석 (2개)
- **로또 방향**
  - 동대문구 로또 당첨 기록
  - SNS 타이밍과 당첨 시기 비교
- **판타지 방향**
  - 이세계 커뮤니티 활동 추적
  - 마지막 게시물 분석
- **스파이 방향**
  - 정보보안 수강 기록
  - 체력 테스트 흔적

### 5단계: 최종 확인 (1-2개)
- 구체적 증거 확보
- 엔딩으로 진입

---

## 경로 2: 인적 네트워크 (가족/지인)

### 1단계: 조사 방향
- "가족에게 연락해보자"

### 2단계: 연락 방법 (3개)
- 어머니께 전화
- 친척 수소문
- 옛 친구 연락

### 3단계: 단서 발견 (3개)
- 매달 50만 원 송금 (→ 선박 방향)
- 2017년 2월 병원 방문 (→ 혼수 방향)
- 경찰서 연락 이력 (→ 목격자 방향)

### 4단계: 추적 (2개)
- **선박 방향**
  - 송금 출처 추적
  - 부산 선박회사 발견
- **혼수 방향**
  - 병원 기록 조회
  - 교통사고 확인
- **목격자 방향**
  - 경찰서 문의
  - 증인보호 프로그램

### 5단계: 최종 확인
- 구체적 위치 파악
- 엔딩으로 진입

---

## 경로 3: 공식 기록 (대학/공공)

### 1단계: 조사 방향
- "대학 기록을 확인하자"

### 2단계: 기록 종류 (3개)
- 학적 기록 (휴학/복학)
- 동아리 활동
- 도서관 대출 기록

### 3단계: 특정 기록 발견 (3개)
- 종교 동아리 가입 (→ 종교 방향)
- 창업 동아리 활동 (→ 사업 방향)
- 법정 출석 기록 (→ 교도소 방향)

### 4단계: 심화 조사 (2개)
- **종교 방향**
  - 절 방문 기록
  - 템플스테이 신청
- **사업 방향**
  - 사업자등록 조회
  - 폐업 신고 확인
- **교도소 방향**
  - 재판 기록 검색
  - 교정시설 확인

### 5단계: 최종 확인
- 구체적 위치/상황 파악
- 엔딩으로 진입

---

## 전체 구조

```
프롤로그
    ↓
1단계: 조사 방향 (3개)
├─ 디지털 흔적
├─ 인적 네트워크
└─ 공식 기록
    ↓
2단계: 세부 방법 (3개)
    ↓
3단계: 특정 단서 (3개)
    ↓
4단계: 심화 분석 (2개)
    ↓
5단계: 최종 확인 (1-2개)
    ↓
엔딩 (9개 중 1개)
```

## 플레이 시간 예상

- 각 단계: 2-3분
- 총 5단계: 10-15분
- 엔딩: 5-10분
- **전체**: 20-30분 (적절)
